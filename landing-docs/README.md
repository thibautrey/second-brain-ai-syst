# Second Brain - Landing & Docs

A dedicated Vercel-ready landing page and documentation site for the Second Brain AI System.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Building

```bash
npm run build
```

The output will be in the `dist` directory, ready to deploy on Vercel.

## Deployment

This project is optimized for Vercel deployment:

1. Push to GitHub
2. Connect repository to Vercel
3. Vercel automatically detects Next.js/Vite configuration
4. Deploy!

No additional configuration needed.

## Structure

- `/src/pages/` - Landing and Docs pages
- `/src/components/ui/` - Reusable UI components
- `/src/i18n/` - Internationalization (EN/FR)
- `/public/` - Static assets
- `vite.config.ts` - Vite configuration
- `tailwind.config.js` - Tailwind CSS configuration

## Technologies

- React 19
- TypeScript
- Tailwind CSS
- Vite
- i18next for translations
- Lucide Icons
